# Usage
<!-- usage -->
```sh-session
$ npm install -g buttercms
$ buttercms COMMAND
running command...
$ buttercms (-v|--version|version)
buttercms/0.0.1 linux-x64 node-v10.5.0
$ buttercms --help [COMMAND]
USAGE
  $ buttercms COMMAND
...
```
