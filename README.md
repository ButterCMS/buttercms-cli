## `buttercms generate:blog`

```
USAGE
  $ buttercms generate:blog

OPTIONS
  --for=express  Target destination for the generator command
```

## `buttercms help [COMMAND]`

display help for buttercms

```
USAGE
  $ buttercms help [COMMAND]

ARGUMENTS
  COMMAND  command to show help for

OPTIONS
  --all  see all commands in CLI
```
